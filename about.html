About The Project

Academic Web Development Project

Welcome to Daily Pulse. This website was designed and developed as a student project to demonstrate proficiency in modern web development technologies.

"The goal of this project was to build a functional, responsive News Application that interacts with a real-world API."

This application fetches live data from The Guardian Open Platform API. It features dynamic DOM manipulation, asynchronous JavaScript (Fetch API), and a custom CSS design inspired by top-tier news publications.

Key Features Implemented:

  • Real-time API Data Fetching
  • Dynamic Category Filtering
  • Search Functionality
  • Responsive Grid Layout
  • Error Handling & Loading States
