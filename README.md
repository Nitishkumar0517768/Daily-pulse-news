## Project Overview

Daily Pulse News is a dynamic web application developed as a student project to demonstrate proficiency in front-end web development. It functions as a real-time news aggregator, fetching and displaying current news articles from a public API.

**This project showcases:**
* Interaction with a real-world REST API.
* Dynamic creation and manipulation of the DOM using pure JavaScript.
* A consistent, responsive user interface across different screen sizes.

## ‚ú® Key Features

* **Real-time Data:** Fetches and displays current news articles using the NewsAPI.
* **Search Functionality:** Allows users to search for news based on keywords.
* **Load More:** Implements pagination to load additional articles on demand.
* **Responsive Design:** Utilizes a custom CSS grid layout for seamless viewing on mobile, tablet, and desktop devices.
* **Consistent UI:** Features a uniform header and footer across the main and about pages.

* **External API:** NewsAPI (for fetching news data).
